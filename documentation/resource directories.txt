This is where Chief and Workers will write files to in their docker containers

chief/app/
	resources/
		modules/
			module-name/
				module-name.tar -> (Processor.tar, Butcher.tar)
				Processor/
					Processor.py
					...
				Butcher/
					Split.py
					Merge.py
			...
	
		jobs/
			job-id/
				module.txt
				data/
					dataset file(s)
					...
				fragment_cache/
					fragment files
					...
				
				result_fragment_cache/
					result fragment files
					...
				results/
					result file(s)
					...
			...
			
worker/app/
	resources/
		modules/
			module-name/
				Processor.py
				...
		
		jobs/
			job-id/
				processor.txt
				fragment_cache/
					fragment files
					...
